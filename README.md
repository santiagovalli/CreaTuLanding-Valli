# Proyecto Ecommerce en ReactJS

Este proyecto es una aplicación de ecommerce creada con ReactJS, utilizando varias dependencias populares para mejorar la experiencia del usuario y gestionar diferentes funcionalidades. A continuación se describen las dependencias utilizadas, las funcionalidades principales y los detalles del proyecto.

## Dependencias Utilizadas

1. **ReactJS**: El framework principal para crear la interfaz de usuario de la aplicación.
2. **Bootstrap**: Un framework de diseño de código abierto utilizado para construir interfaces responsivas y modernas.
   - **Link oficial**: [https://getbootstrap.com/](https://getbootstrap.com/)
3. **SweetAlert2**: Una librería para mostrar alertas personalizadas y modales en la aplicación.
   - **Link oficial**: [https://sweetalert2.github.io/](https://sweetalert2.github.io/)
4. **Firebase**: Se utiliza para la autenticación de usuarios y el almacenamiento de datos (como productos y pedidos) en tiempo real.
   - **Link oficial**: [https://firebase.google.com/](https://firebase.google.com/)
5. **React Router**: Para manejar la navegación entre diferentes vistas o páginas dentro de la aplicación.
   - **Link oficial**: [https://reactrouter.com/](https://reactrouter.com/)

## Funcionalidades Principales

### 1. Presentación de Productos

La aplicación muestra una lista de productos que los usuarios pueden explorar. Los productos se presentan de forma dinámica a partir de la base de datos, permitiendo que el contenido se actualice en tiempo real.

### 2. Filtrar por Categoría

Los usuarios pueden filtrar los productos según diferentes categorías. Esto permite una búsqueda más rápida y precisa según las preferencias de los usuarios.

### 3. Carrito de Compras

La aplicación incluye un carrito de compras donde los usuarios pueden agregar productos, ver los detalles de su compra y actualizar la cantidad de artículos.

### 4. Formulario de Compra

Los usuarios pueden completar un formulario con sus datos personales para proceder a la compra, proporcionando información como nombre, dirección de envío y detalles de pago.

### 5. Finalizar la Compra

Una vez que el usuario haya completado el formulario, puede proceder a la compra finalizando el pedido. Esto incluye la confirmación de la compra y el envío de los detalles a la base de datos de Firebase.
